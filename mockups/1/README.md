# Investment Dashboard Mockup

Open `dashboard_mockup.html` in a browser.

This is a static visual mockup for the private Windows investment app dashboard.
It includes:
- Default icon in the top-left
- Market/filter dropdown
- AI status cards
- £1-£10 share candidate table
- AI suggestions
- Sector/index snapshot
- AI activity log

The icon is embedded into the HTML as base64, so the file can be opened by itself.
